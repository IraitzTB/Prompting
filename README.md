# Prompting

Este proyecto es una librería diseñada para realizar cálculos matemáticos de manera sencilla y eficiente. Incluye funciones y utilidades para facilitar operaciones comunes y avanzadas.

# Contenido

- [Instalación](#instalación)
- [Uso](#uso)
- [Contribuciones](#contribuciones)
- [Licencia](#licencia)

## Instalación

Para instalar la librería, simplemente ejecute el siguiente comando:

```
pip install mi_libreria_calculos
```

## Uso

Aquí hay algunos ejemplos de cómo usar la librería:

```python
from mi_libreria_calculos import suma, resta

resultado_suma = suma(5, 3)
resultado_resta = resta(10, 4)
```
